# 📝 Todo List

A simple and beginner-friendly Todo List application built using **HTML**, **CSS**, and **JavaScript**. This project allows users to add and delete tasks with an easy-to-use interface.

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✅ Simple and responsive design
- 📱 Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
